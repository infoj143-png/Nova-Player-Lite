# APK Download Guide

Follow these steps to download and install the latest debug version of Nova Player Lite from GitHub.

## 📥 How to Download the APK

1.  **Navigate to the GitHub Actions Tab**: Go to the main page of the Nova-Player-Lite repository and click on the **Actions** tab at the top.
2.  **Select the Latest Workflow Run**: Look for the most recent run named "Android Build" (it should have a green checkmark if it succeeded). Click on the run title.
3.  **Find the Artifacts Section**: Scroll down to the bottom of the run summary page to find the **Artifacts** section.
4.  **Download the APK**: Click on **NovaPlayerLite-Debug-APK**. This will download a ZIP file containing the APK to your computer or device.
5.  **Extract the ZIP**: Extract the downloaded ZIP file to get the `app-debug.apk` file.

## 📲 How to Install the APK on Android

1.  **Transfer the APK**: If you downloaded the APK on a computer, transfer the `app-debug.apk` file to your Android device via USB, email, or a cloud service.
2.  **Enable Unknown Sources**:
    -   On most modern Android versions, you will be prompted to allow the installation from the specific app you are using (e.g., Chrome or File Manager).
    -   If not prompted, go to **Settings > Security** (or **Apps & notifications > Special app access**) and enable **Install unknown apps** for the app you're using to open the APK.
3.  **Locate and Install**:
    -   Open your device's **File Manager** app.
    -   Find the `app-debug.apk` file.
    -   Tap on the file and select **Install**.
4.  **Open the App**: Once installed, you can find Nova Player Lite in your app drawer and open it!

---
*Note: This is a debug build and may not be suitable for production use. Ensure you trust the source before installing APKs from outside the Google Play Store.*
